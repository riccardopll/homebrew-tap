# riccardopll's Homebrew Tap

Homebrew tap for shipping my CLI tools.

## Install

```bash
brew tap riccardopll/tap
brew install riccardopll/tap/<name>
```

## Formulae

| Formula  | Description                    |
| -------- | ------------------------------ |
| `ical`   | Read-only macOS Calendar CLI   |
| `ytfeed` | Scrape YouTube homepage videos |
