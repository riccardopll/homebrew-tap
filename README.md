# Riccardopll Tap

## How do I install these formulae?

`brew install riccardopll/tap/<formula>`

Or `brew tap riccardopll/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "riccardopll/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
